# machitools
マチアプリで使用するAPIサーバーサイドです。

# チェックアウト

    $ goapp get github.com/keima/machiasobi-tools

これでOKだと思います。

ダメだったら、`$GOPATH/src/github.com/keima/machiasobi-tools` にgit cloneする感じです。

# 試す

    $ goapp serve machitools

# デプロイ

    $ cd ./machitools
    $ goapp deploy app-prod.yaml
